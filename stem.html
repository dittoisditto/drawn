Video Game Design 2019-2021

Video Game Design was one of the events that I had signed up for for 2020. This was the first time I had ever tried game design, so I wasn't confident in my ability to program the game. Instead, I decided to use my art/animation ability to create the assets for the game.

The theme for the event was the 2020 Tokyo Olympics. The game idea that my team and I had was a 4 stage game, the first three stages being "training stages", and the last stage being the final race between the main character and other NPCs. The player would spend most of their time in the first three stages trying to build their skills in order to win the final race.

3D Animation 2019-2020

3D Animation was the other event that I had signed up for for TSA 2020. I had had a little more experience with 3D Animation, particularly with regards to 3D modelling. Thus, I was the main 3D modeller for my event group. I also did the main designs for the characters, as shown below in the form of reference sheets.

The theme for the event was human impact on the environment. In short, the main character of the animation (the baby fox) would go through three stages of life. Each stage of life would depict more environmental damages, and it would slowly have a greater impact on the main character's health. This progression would be used to indicate how human behavior could negatively impact animals, to the point where the main character was no longer healthy enough to survive in his environment.

Game Design

Liberating Infection

During the summer of 2020, I decided to try learning game design after reading a few articles about a game engine called Godot. I used the knowledge I gained from tutorials and articles to create a game called Liberating Infection. While the game itself is very rudimentary, it helped me learn a lot about game design and using Godot as a game engine. Below is a link to the itch.io page, as well as screenshots of the game.

I had originally intended for the game to be a short, yet informative game about proper hygiene during the pandemic. However, I realized that there was a lot about game design that I had not learned about, thus I wasn't at the skill level to make a developed game like that. Instead, I used the game as more of a learning curve, implementing as much interesting features as I could within it. While the game itself isn't very fun to play, it taught me a lot about how game design works and how to implement a lot of different features that I will definitely use for future games.

Middle Grade STEM

As someone that aspired to work in STEM, I always tried to find as many learning resources as possible. However, I've found that it can be very difficult to find people that teach about STEM related topics like Website Design and Python. Therefore, I decided that I wanted to help other people who have had to go through this same struggle when starting STEM. In the summer of 2020, I teamed up with other students to form Middle Grade STEM, a free volunteer tutoring program that gives middle school students teachers and resources to learn as much as they possibly can.
